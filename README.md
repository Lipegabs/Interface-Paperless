# PaperLess Dashboard

Projeto desenvolvido para praticar HTML e CSS através da recriação de uma interface de gerenciamento de documentos inspirada em um protótipo de dashboard.

## Tecnologias utilizadas

* HTML5
* CSS3
* Flexbox
* CSS Grid

## Objetivos do projeto

* Praticar estruturação semântica com HTML.
* Desenvolver layouts utilizando Flexbox e Grid.
* Aprender a organizar componentes de interface.
* Reproduzir visualmente um dashboard moderno.

## Funcionalidades

* Menu lateral de navegação.
* Área principal de documentos.
* Campo de busca.
* Cards de documentos.
* Layout responsivo básico.

## Aprendizados

Durante o desenvolvimento deste projeto foram praticados conceitos como:

* Estrutura semântica (`header`, `nav`, `main`, `section`, `aside`);
* Organização de componentes;
* Posicionamento com Flexbox;
* Criação de grades com CSS Grid;
* Estilização de interfaces modernas.
